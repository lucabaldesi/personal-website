NetworkX is a Python package for the creation, manipulation, and study of the structure, dynamics, and functions of complex networks. I worked on the spectral graph forge and the internet autonomous system network generators.

Webpage: NetworkX website

Peerstreamer

PeerStreamer is an open source P2P Media Streaming framework written in C, developed as part of the NAPA-WINE European research project. It includes a streaming engine for the efficient distribution of media streams, a source application for the creation of channels and a player applications to visualize the streams. I designed and implemented most of the software besides coordinating occasional contributors.

Webpage: Peerstreamer website
Period: April 2013 - December 2018
Founding: ANS group
